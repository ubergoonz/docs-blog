# :material-package-variant-plus: How to install

The official install guide is available at [https://ghostty.org/docs/install/binary](https://ghostty.org/docs/install/binary){target="_blank"}

## Install on macOS :simple-macos: using Homebrew :simple-homebrew:

=== "To install"

    ```bash title="Install on macOS using Homebrew"

    brew install --cask ghostty

    ```

=== "After install"

    ```bash
    which ghostty
    ```
    !!! success

        `which ghostty` should show {++`/Applications/Ghostty.app/Contents/MacOS/ghostty`++}
