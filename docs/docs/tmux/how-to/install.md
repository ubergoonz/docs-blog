# :material-package-variant-plus: How to install

The official install guide is available at [https://github.com/tmux/tmux/wiki/Installing](https://github.com/tmux/tmux/wiki/Installing){target="_blank"}.

## Install on macOS :simple-macos: using Homebrew :simple-homebrew:

=== "To install"

    ```bash title="Install on macOS using Homebrew"

    brew install tmux

    ```

=== "After install"

    ```bash
    which tmux
    ```
    !!! success

        `which tmux` should show {++`/opt/homebrew/bin/tmux`++}
