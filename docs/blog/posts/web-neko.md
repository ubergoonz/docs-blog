# Web Neko Cat

If you are looking for a fun and interactive way to pass the time, look no further than [Web Neko](https://webneko.net){target="_blank"}! Web Neko is a virtual cat that lives on your web browser. You can interact with it by petting it, feeding it, and even playing games with it. It's a great way to relax and have some fun during your breaks while browsing the web. Give it a try and see how much fun you can have with your new virtual feline friend!

## To use it in your own website
You can easily embed Web Neko into your own website by adding the following code snippet to your HTML:

```html
<script>NekoType="white"</script>
<h1 id=nl><script src="https://webneko.net/n20171213.js"></script><a 
href="https://webneko.net">Neko</a></h1>
```

What I did is add it into the `index.md` in the root directory of this website, so you can see it in action right here on this site! Feel free to explore and have fun with Web Neko!
